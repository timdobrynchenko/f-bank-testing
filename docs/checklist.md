## Сводка по статусам

**Passed: 13** — функционал работает по спецификации.
**Error: 7** — найденные дефекты:
- №6: поле принимает 17 цифр (см. [bug-03](bug-reports/bug-03-card-length.md))
- №10: перевод 0 руб. принимается банком
- №11: непоследовательная валидация при balance=0
- №12: принимается отрицательная сумма (см. [bug-02](bug-reports/bug-02-negative-amount.md))
- №15: e-нотация в комиссии при огромных числах
- №17: некорректное округление комиссии (99 → 0 вместо 9) (см. [bug-01](bug-reports/bug-01-commission-rounding.md))
- №18: некорректное округление комиссии (граничный случай 91)
- №20: при смене валюты не пересчитывается состояние формы
